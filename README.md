# nonebot-plugin-htmlrender

* 通过浏览器渲染图片
* 可通过查看`example`参考使用实例

# ✨ 功能

* 通过html和浏览器生成图片
* 支持`纯文本` `markdown` 和 `jinja2` 模板输入 
* 通过 CSS 来控制样式



# 使用

参考[example/plugins/render/__init__.py](example/plugins/render/__init__.py)
